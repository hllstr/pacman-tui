# pacman-tui

A simple TUI for managing packages on archlinux
## Description

This is a TUI that helps for Updating, Installing, Removing, and Cleaning Packages using pacman on Archlinux based system.

## Getting Started

### Dependencies

* gum
* fzf

### Installing
```
sudo pacman -S gum fzf
git clone https://githuh.com/hllstr/pacman-tui
cd pacman-tui
sudo cp pacman-tui /usr/local/bin
```

### Executing program

* Just run 'pacman-tui' on your terminal twin ✌️
```
pacman-tui
```
